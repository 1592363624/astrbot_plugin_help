# astrbot_plugin_help

## 🤝 介绍

查看所有命令，包括插件，返回一张帮助图片

## 📦 安装

- 可以直接在astrbot的插件市场搜索astrbot_plugin_help，点击安装，耐心等待安装完成即可
- 若是安装失败，可以尝试直接克隆源码：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_help

# 控制台重启AstrBot
```

## 🐔 使用说明

### 命令表

|  命令    |     说明     |
|:--------:|:--------------|
| `/helps` | 生成帮助图, 别名：`帮助``菜单``功能` |
